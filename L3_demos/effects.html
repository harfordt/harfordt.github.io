L3 Web Demos

Level 3 Web development demos

This site contains a bunch of demos of concepts related to L3 web development. Each page has a link to the source and instructions on how to implement.

Most plugins require you to:

  • Copy some .css and .js files and link them in your .html file.
  • Maybe add some jquery of your own (either in script tags or a separate .js file if you want the code on multiple pages
  • Structure your html in a specific way (using certain ids and/or classes).
